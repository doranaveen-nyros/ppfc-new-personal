
The SPPFC (Sri Padma Priya Finance Company) system is a comprehensive financial and operational management platform designed to streamline core processes in a multi-branch hire purchase (HP) and vehicle finance business. The system enables centralized control over:

- Master data management (branches, areas, consultants, agents)
- Financial operations (receipts, capital entries, PDR locks, expense tracking)
- Reporting & analytics (collection, recovery, RC status, NOC dispatch)
- Compliance & audit (closure tracking, NOC generation)

| Area   |      Tech      |
|----------|-------------|
| Programming language |  C# |
| Frontend |    .Net Blazor Server App   |
| Backend |    .NET Core Web API   |
| API Gateway | Ocelot |
| Database |    MS SQL   |
| Authentication | JWT |
| APIs | RESTful APIs for all modules |
| File Storage | AWS S3 or local disk for uploaded images/docs |
| Version Control | Git + GitHub |

## How to Setup in Local
- Open Solution Visual Studio 2022
- Build the Solution
- Configure Startup Project - set multiple project
- Run the Solution
